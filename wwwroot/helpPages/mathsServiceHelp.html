Service /api/maths - Documentation

Liste de query string possible

L'addition

Exemple d'utilisation : ?op=+&x=4&y=4

Retourne : {"op":"+", "x":nombre, "y":nombre, "value":4 + 4}

La soustraction

Exemple d'utilisation : ?op=-&x=4&y=4

Retourne : {"op":"-", "x":nombre, "y":nombre, "value":4 - 4}

La multiplication

Exemple d'utilisation : ?op=*9&x=4&y=4

Retourne : {"op":"+", "x":nombre, "y":nombre, "value":4 * 4}

Le modulo

Exemple d'utilisation : ?op=%&x=4&y=4

Retourne : {"op":"+", "x":nombre, "y":nombre, "value":4 % 4}

Le factoriel

Exemple d'utilisation : ?op=!&x=4

Retourne : {"op":"+", "x":nombre, "value":4!}

Le nombre premier

Exemple d'utilisation : ?op=p&x=nombre

Retourne : {"op":"+", "x":nombre, "value":vrai si le nombre est un nombre premier}

La valeur du nième nombre premier

Exemple d'utilisation : ?op=np&x=nombre

Retourne : {"op":"+", "x":nombre, "value":le nième nombre premier}
